## Why Use This Library?

* Zero dependencies
* Doesn't require the DOM
* Render SVG charts inside of React Server Components!
* Amazing TypeScript Developer Experience

## Render chats in...

* React / React Native
* Express.js
* Any other JavaScript environment

## Installation

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>
